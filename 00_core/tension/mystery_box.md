# Règle : Mystery Box (La Machine à Curiosité)

## Définition
Énigme centrale ou information retenue qui stimule la curiosité du lecteur (concept J.J. Abrams). La **question** importe souvent plus que la réponse. Le cerveau humain est câblé pour résoudre les mystères.

## Connexions Systèmes
- **Dépendance Amont** : `00_core/spec/04_premise.md` (Le Hook EST souvent une Mystery Box)
- **Dépendance Aval** : `dramatic_irony.md` (Le mystère peut être connu du lecteur, pas du personnage)

## Algorithme Fondamental

```
QUESTION POSÉE (Ouverture) + INDICES PARSEMÉS (Delay) + RÉPONSE SATISFAISANTE (Payoff)
```

La Mystery Box crée un "itch cognitif" que le lecteur veut gratter.

## Livrables Strictes (Output attendu)

### Template de Mystery Box

| Composante | Question | Exemple (Lost) |
|---|---|---|
| **QUESTION** | Quel est le mystère central ? | "Qu'est-ce que l'île ?" |
| **POSE** | Comment la question est-elle introduite ? | Crash d'avion sur une île impossible |
| **INDICES** | Quels éléments nourrissent le mystère ? | Chiffres, Dharma, fumée noire |
| **RÉPONSE** | La réponse est-elle prévue/satisfaisante ? | ⚠️ Controversée (Lost) |
| **IMPACT** | La réponse transforme-t-elle la compréhension ? | La question était plus forte que la réponse |

### Les 3 Règles de la Mystery Box

1. **La Question doit être claire** : Le lecteur doit savoir qu'il y a un mystère à résoudre.
2. **Les Indices doivent nourrir** : Chaque nouvel élément doit enrichir le mystère, pas le diluer.
3. **La Réponse doit honorer** : Même simple, elle doit respecter les règles établies.

## Anti-patterns (Motifs de Rejet Immédiat)

- ❌ **Mystery Box Sans Réponse** : Le mystère n'est jamais résolu. (Frustration maximale.)
- ❌ **Réponse Décevante** : La réponse trahit l'attente créée. (Le contrat est rompu.)
- ❌ **Trop de Boxes** : Multiplication des mystères non résolus. (Confusion.)
- ❌ **Question Floue** : Le lecteur ne sait pas ce qu'il est censé chercher.
- ❌ **Deus Ex Machina** : La réponse sort de nulle part, non préparée.

## Application pour l'Audit (Tests)

1. **Test de la Clarté** : Le lecteur sait-il qu'il y a un mystère central ?
2. **Test de l'Engagement** : Le mystère reste-t-il intrigant au fil du récit ?
3. **Test de la Résolution** : La réponse est-elle cohérente avec les indices donnés ?
4. **Test du Fair Play** : Un lecteur attentif pouvait-il deviner (ou presque) ?
