# Règle : Motif Recurrence (Récurrence de Motif)

## Définition
Répétition délibérée d'images, objets, phrases ou situations à travers le récit pour renforcer un thème sous-jacent et créer des connexions symboliques.

## Principe
Un motif isolé est un détail. Un motif répété devient un symbole. Un symbole qui évolue porte le thème.

## Types de Motifs
- **Visuel** : Une couleur, un objet, un lieu.
- **Verbal** : Une phrase, un mot, un dialogue.
- **Situationnel** : Une action, un rituel, un type de scène.
- **Sonore** : Un bruit, une musique (dans le cas de scripts/scénarios).

## Évolution
Le motif doit évoluer avec le personnage. Sa signification change au fil du récit.

## Application pour l'Audit
- **Identification** : Quels éléments réapparaissent à travers le récit ?
- **Évolution** : Ces motifs changent-ils de signification ou restent-ils statiques ?
- **Lien thématique** : Les motifs renforcent-ils la Controlling Idea ?
