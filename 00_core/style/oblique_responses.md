# Règle : Oblique Responses (Réponses Obliques)

## Définition
Les personnages répondent partiellement, indirectement ou à côté de la question, créant une texture réaliste et du sous-texte.

## Principe
Dans la vraie vie, les gens répondent rarement directement. Ils évitent, détournent, répondent à ce qu'ils auraient voulu entendre, ou ignorent la question.

## Types de Réponses Obliques
- **Évitement** : Répondre par une autre question.
- **Détournement** : Changer de sujet.
- **Réponse partielle** : Ne répondre qu'à une partie.
- **Non-réponse** : Action au lieu de mots.
- **Réponse différée** : Répondre à une question posée plus tôt.

## Exemple
> "Tu m'aimes encore ?"
> "Le café est prêt."

## Application pour l'Audit
- **Réalisme** : Les personnages répondent-ils toujours trop directement ?
- **Sous-texte** : Les non-réponses révèlent-elles quelque chose de caché ?
- **Variété** : Y a-t-il une diversité dans les types de réponses ?
