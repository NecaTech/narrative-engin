# Règle : Active Voice (Voix Active)

## Définition
Privilégier la voix active (sujet → verbe → objet) sauf intention stylistique délibérée, pour une prose plus directe et dynamique.

## Principe
- **Voix active** : "Le loup dévora l'agneau." (Direct, puissant)
- **Voix passive** : "L'agneau fut dévoré par le loup." (Distant, faible)

## Quand utiliser le passif
- Pour mettre l'accent sur l'objet plutôt que l'acteur.
- Quand l'acteur est inconnu ou non pertinent.
- Pour créer une distance émotionnelle délibérée.

## Impact
- Active = énergie, clarté, engagement.
- Passive = distance, ambiguïté, ralentissement.

## Application pour l'Audit
- **Détection** : Y a-t-il un excès de constructions passives non justifiées ?
- **Justification** : Chaque passif a-t-il une raison d'être ?
- **Dynamisme** : La prose semble-t-elle énergique ou molle ?
