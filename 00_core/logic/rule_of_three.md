# Règle : Rule of Three (Le Pattern Ternaire)

## Définition
Principe selon lequel les éléments présentés par **groupes de trois** sont plus satisfaisants, mémorables et efficaces. Trois est le minimum pour établir un pattern et le subvertir.

## Connexions Systèmes
- **Dépendance Amont** : `00_core/structure/try_fail_cycles.md` (2 échecs + 1 succès)
- **Dépendance Aval** : `00_core/tension/foreshadowing.md` (3 indices = pattern établi)

## Algorithme Fondamental

```
ÉLÉMENT 1 (Setup) + ÉLÉMENT 2 (Confirmation) + ÉLÉMENT 3 (Payoff/Subversion)
```

Le troisième élément peut **confirmer** le pattern ou le **subvertir**.

## Livrables Strictes (Output attendu)

### Applications de la Règle de 3

| Domaine | Pattern de 3 | Exemple |
|---|---|---|
| **Structure** | 3 actes | Début / Milieu / Fin |
| **Échec** | 2 échecs + 1 réussite | Essai-Échec x2 → Victoire |
| **Humour** | Setup + Setup + Punchline | "Un prêtre, un rabbin, et un imam..." |
| **Suspense** | 2 fausses alertes + 1 vraie | Craquement (rien) + Craquement (chat) + Craquement (tueur) |
| **Rhétorique** | 3 arguments | "Liberté, Égalité, Fraternité" |
| **Description** | 3 détails | "Grand, voûté, et ses yeux ne clignaient jamais." |

### Pourquoi 3 Fonctionne

1. **Minimum pour un pattern** : 1 = hasard, 2 = coïncidence, 3 = pattern.
2. **Rythme naturel** : Le cerveau attend le troisième temps.
3. **Subversion possible** : Le 3ème peut surprendre.

## Anti-patterns (Motifs de Rejet Immédiat)

- ❌ **Pattern de 2** : Setup + Payoff sans confirmation. (Trop abrupt.)
- ❌ **Pattern de 4+** : Trop long, le lecteur décroche.
- ❌ **3ème Identique** : Les 3 éléments sont pareils. (Pas de progression.)
- ❌ **Subversion Forcée** : Le 3ème est bizarre juste pour surprendre.

## Application pour l'Audit (Tests)

1. **Test du Comptage** : Les éléments importants sont-ils présentés par 3 ?
2. **Test du Rythme** : Le récit utilise-t-il des triplets pour créer du rythme ?
3. **Test du Payoff** : Le 3ème élément apporte-t-il confirmation ou subversion ?
4. **Test de l'Escalade** : Les 3 éléments progressent-ils en intensité ?
