# Règle : Try-Fail Cycles (Cycles d'Essai-Échec)

## Définition
Le protagoniste échoue plusieurs fois avant de réussir, chaque échec enseignant quelque chose de nécessaire à la victoire finale.

## Principe
La victoire facile est insatisfaisante. Les échecs successifs :
- Augmentent les enjeux.
- Révèlent le caractère du héros.
- Rendent la victoire finale méritée.

## Structure typique
1. **Premier essai** : Échec par ignorance.
2. **Deuxième essai** : Échec par orgueil ou erreur.
3. **Troisième essai** : Succès grâce aux leçons apprises.

## Application pour l'Audit
- **Échecs présents** : Le héros rencontre-t-il de vrais obstacles ?
- **Apprentissage** : Chaque échec apporte-t-il une leçon visible ?
- **Victoire méritée** : Le succès final utilise-t-il les leçons des échecs ?
