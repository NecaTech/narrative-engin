# Règle : Save the Cat (La Méthode du Battement)

## Définition
Une méthode de structuration narrative basée sur une "Beat Sheet" (feuille de route) de 15 points clés. Elle vise à équilibrer le rythme émotionnel et l'action pour garder l'audience engagée.

## Points Clés (Exemples)
1. **L'Image d'Ouverture** : Un aperçu de l'ancien monde avant le changement.
2. **Le Thème Narratif** : Ce que le héros doit apprendre.
3. **Le Catalyseur** : L'incident déclencheur qui change tout.
4. **Le Débat** : Le moment où le héros hésite.
5. **Le Passage à l'Acte II** : Le héros quitte son monde familier.

## Application pour l'Audit
- Le chapitre respecte-t-il le "Beat" prévu dans la structure globale ?
- L'évolution de l'intrigue est-elle trop lente ou trop rapide par rapport au plan ?
