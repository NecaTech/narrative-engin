# Règle : Want vs Need (Le Moteur Psychologique)

## Définition
Système binaire de motivation du personnage. Le **Want** est l'objectif visible (fausse solution). Le **Need** est la leçon cachée (vraie solution). L'histoire force le personnage à choisir entre les deux.

## Connexions Systèmes
- **Dépendance Amont** : `ghost_wound_lie.md` (Le Lie génère le Want)
- **Dépendance Aval** : `00_core/arc/positive_arc.md` (L'arc consiste à abandonner le Want pour le Need)
- **Audit Applicable** : `03_audit/spec/06_characters_audit.md`

## Algorithme Fondamental

```
LIE → WANT (Fausse Solution) ≠ NEED (Vraie Solution) → CONFLIT INTERNE
```

- Le **Want** est ce que le personnage CROIT devoir faire pour guérir sa Wound.
- Le **Need** est ce qu'il doit RÉELLEMENT faire (souvent l'inverse).
- Le Climax force un choix : **Want OU Need** (pas les deux).

## Livrables Strictes (Output attendu)

### Template Obligatoire

| Composante | Question | Exemple (Michael Corleone) |
|---|---|---|
| **WANT (Visible)** | Quel objectif externe poursuit-il ? | Protéger sa famille, venger son père |
| **NEED (Caché)** | Quelle vérité doit-il accepter ? | S'éloigner du crime pour sauver son âme |
| **INCOMPATIBILITÉ** | Pourquoi ne peut-il pas avoir les deux ? | Protéger la famille = devenir le monstre qu'il fuyait |
| **CHOIX AU CLIMAX** | Que choisit-il ? (Want ou Need ?) | Want → Arc Négatif (il devient Don) |

### Règles de Construction

1. **Le Want doit être CONCRET** : Objectif physique, mesurable. ("Tuer le dragon", pas "trouver la paix".)
2. **Le Need doit être ABSTRAIT** : Leçon émotionnelle, croissance interne. ("Accepter l'aide", "Pardonner".)
3. **L'incompatibilité doit être LOGIQUE** : Le personnage ne peut pas avoir les deux sans contradiction.

## Anti-patterns (Motifs de Rejet Immédiat)

- ❌ **Want Abstrait** : "Il veut être heureux." (C'est un Need déguisé. Le Want doit être un OBJECTIF tangible.)
- ❌ **Need Évident** : Le personnage sait dès le début ce dont il a besoin. (Le Need doit être une RÉVÉLATION.)
- ❌ **Conflit Inexistant** : Want et Need peuvent coexister sans sacrifice. (Pas de tension = pas d'histoire.)
- ❌ **Climax sans Choix** : Le personnage obtient les deux par chance. (Le Climax DOIT forcer un sacrifice.)
- ❌ **Need Imposé** : Un mentor dit au héros ce dont il a besoin. (Le héros doit le DÉCOUVRIR par la souffrance.)

## Application pour l'Audit (Tests)

1. **Test du Poster** : Peut-on résumer le Want en 5 mots sur une affiche de film ? Si non, il n'est pas assez concret.
2. **Test du Choix Cornélien** : Au Climax, le personnage doit-il renoncer à quelque chose pour gagner ?
3. **Test du "All is Lost"** : Le moment de crise révèle-t-il le Need au personnage ?
4. **Test de l'Arc** : Le personnage finit-il l'histoire différent de comment il l'a commencée ?
