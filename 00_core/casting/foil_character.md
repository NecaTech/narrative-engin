# Règle : Foil Character (Personnage Faire-Valoir)

## Définition
Un personnage conçu pour contraster avec un autre (souvent le protagoniste), mettant en relief ses qualités, ses défauts ou ses choix par opposition.

## Principe
Le contraste rend visible ce qui pourrait rester implicite. En voyant le Foil agir différemment, le lecteur comprend mieux le protagoniste.

## Types de Foil
- **Philosophique** : Visions du monde opposées.
- **Comportemental** : Réactions opposées aux mêmes situations.
- **Moral** : Choix éthiques divergents.
- **Complémentaire** : Forces et faiblesses inversées.

## Distinction avec Antagonist Mirror
- **Foil** : Peut être un allié, un ami, un neutre. Le contraste est la fonction principale.
- **Antagonist Mirror** : Toujours un opposant. Le miroir sert le conflit.

## Application pour l'Audit
- **Présence** : Y a-t-il un personnage qui contraste explicitement avec le protagoniste ?
- **Fonction** : Ce contraste révèle-t-il quelque chose sur le héros ?
- **Équilibre** : Le Foil est-il un personnage à part entière ou juste un outil ?
