# Règle : False Protagonist (Le Leurre Narratif)

## Définition
Technique de subversion où le personnage présenté comme héros central est **éliminé ou marginalisé** après avoir capté l'attention du lecteur. Le "vrai" protagoniste émerge ensuite. Utilisé pour créer un choc et redéfinir les enjeux.

## Connexions Systèmes
- **Dépendance Amont** : Aucune (Technique structurelle indépendante)
- **Dépendance Aval** : `00_core/structure/in_medias_res.md` (Souvent combiné avec une ouverture choc)
- **Audit Applicable** : `03_audit/narrative/plot_hole_tracker.md` (Vérifier la cohérence post-révélation)

## Algorithme Fondamental

```
SETUP (Investissement dans le Faux) → CHOC (Élimination/Révélation) → PAYOFF (Le Vrai incarne mieux le Thème)
```

Le faux protagoniste doit être **suffisamment développé** pour que sa perte crée un impact, mais pas au point d'éclipser le vrai héros.

## Livrables Strictes (Output attendu)

### Checklist Obligatoire

| Composante | Question | Exigence |
|---|---|---|
| **INVESTISSEMENT** | Le lecteur s'est-il attaché au faux protagoniste ? | Minimum 10-20% du récit consacré à lui |
| **INDICES** | Des signes subtils préparent-ils le switch ? | Au moins 2 indices relisibles après coup |
| **TRANSITION** | Le vrai protagoniste est-il présent AVANT le switch ? | Oui, même si en arrière-plan |
| **JUSTIFICATION** | Le switch sert-il le thème ou l'intrigue ? | Pas de choc gratuit |
| **INTÉRÊT** | Le vrai héros est-il aussi (ou plus) intéressant ? | Obligatoire |

### Types de Faux Protagoniste

1. **La Mort Choc** : Le faux meurt brutalement. (*Ned Stark, Psychose*)
2. **Le Révélé Secondaire** : On découvre qu'il n'était qu'un témoin. (*Dr Watson*)
3. **Le Méchant Déguisé** : Le "héros" était l'antagoniste depuis le début.
4. **Le Passeur de Flambeau** : Il transmet sa mission au vrai héros avant de disparaître.

## Anti-patterns (Motifs de Rejet Immédiat)

- ❌ **Choc Gratuit** : Le switch ne sert aucun propos narratif ou thématique. (Le lecteur se sent trahi.)
- ❌ **Vrai Héros Fade** : Le vrai protagoniste est moins intéressant que le faux. (Erreur fatale.)
- ❌ **Zéro Préparation** : Aucun indice n'annonçait le switch. (Lecture "triché".)
- ❌ **Transition Brutale** : Le lecteur n'a pas le temps de s'attacher au vrai héros. (Détachement émotionnel.)
- ❌ **Resurrection** : Le faux protagoniste revient finalement. (Annule l'impact.)

## Application pour l'Audit (Tests)

1. **Test du "Fair Play"** : En relisant, les indices du switch sont-ils visibles mais non évidents ?
2. **Test de l'Attachement** : Le lecteur ressent-il une perte réelle à l'élimination du faux ?
3. **Test du Transfert** : Le vrai héros reprend-il ou transcende-t-il la mission du faux ?
4. **Test du Thème** : Le switch illustre-t-il le thème central ?

## Référence Théorique
- Alfred Hitchcock, *Psychose* (Analyse du faux protagoniste)
