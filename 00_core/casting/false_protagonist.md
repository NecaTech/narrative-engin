# Règle : False Protagonist (Faux Protagoniste)

## Définition
Un personnage présenté initialement comme le héros central, puis écarté (mort, disparition, révélation), subvertissant les attentes du lecteur.

## Principe
Créer un choc narratif en brisant le contrat implicite avec le lecteur. Le "vrai" protagoniste émerge après la disparition du faux.

## Effets
- **Choc émotionnel** : Le lecteur était investi dans le faux protagoniste.
- **Tension accrue** : "Si lui peut mourir, personne n'est en sécurité."
- **Révélation thématique** : Le vrai héros incarne mieux le thème.

## Mise en garde
- Technique risquée : peut aliéner le lecteur si mal exécutée.
- Le "vrai" protagoniste doit être aussi (ou plus) intéressant.

## Application pour l'Audit
- **Justification** : La subversion sert-elle le récit ou est-elle gratuite ?
- **Transition** : Le lecteur peut-il s'attacher au vrai protagoniste ?
- **Préparation** : Des indices subtils préparaient-ils ce retournement ?
