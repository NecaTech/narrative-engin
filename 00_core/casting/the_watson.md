# Règle : Le "Watson"

## Définition
Un personnage (souvent secondaire) dont la fonction narrative est de poser les questions que le lecteur se pose, permettant d'exposer le lore, les règles du monde ou les motivations de manière naturelle.

## Principe
Inspiré du Dr Watson dans les récits de Sherlock Holmes. Watson n'est pas stupide ; il représente le lecteur intelligent mais non-expert. Ses questions évitent les expositions lourdes du narrateur.

## Application pour l'Audit
- **Naturalité de l'exposition** : L'information sur le monde est-elle révélée via un dialogue crédible ou via un monologue artificiel ?
- **Fonction du personnage** : Y a-t-il un personnage désigné (ou plusieurs) pour porter les questions du lecteur ?
- **Éviter le "As You Know, Bob"** : Deux experts ne doivent pas s'expliquer mutuellement des choses qu'ils connaissent déjà.
