# Règle : Ghost/Wound/Lie (Fantôme/Blessure/Mensonge)

## Définition
Triptyque fondamental de la psychologie du personnage : un traumatisme passé (Ghost) qui crée une blessure émotionnelle (Wound), laquelle génère une croyance limitante (Lie) que l'arc narratif doit résoudre.

## Les Trois Composantes
- **Ghost** : L'événement traumatique du passé (abandon, échec, trahison).
- **Wound** : La cicatrice émotionnelle qui en résulte (peur, honte, colère).
- **Lie** : La fausse croyance que le personnage adopte pour se protéger ("Je ne mérite pas l'amour", "Faire confiance = souffrir").

## Lien avec Want vs Need
- Le **Lie** empêche le personnage de voir son vrai **Need**.
- L'arc narratif consiste à confronter le **Lie** pour atteindre la **Truth**.

## Application pour l'Audit
- **Définition claire** : Le Ghost/Wound/Lie de chaque personnage principal est-il documenté ?
- **Impact visible** : Le Lie influence-t-il les décisions du personnage dans l'histoire ?
- **Résolution** : L'arc narratif confronte-t-il et résout-il (ou non) le Lie ?
