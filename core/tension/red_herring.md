# Règle : Red Herring (Fausse Piste)

## Définition
Un indice trompeur introduit délibérément pour détourner l'attention du lecteur de la véritable résolution.

## Principe
Le Red Herring doit être suffisamment crédible pour tromper sans être injuste. Le lecteur doit pouvoir, rétrospectivement, voir qu'il s'agissait d'une fausse piste.

## Distinction importante
- **Red Herring intentionnel** : Technique légitime de misdirection.
- **Red Herring accidentel** : Élément introduit sans résolution (voir Chekhov's Gun) — c'est un défaut.

## Application pour l'Audit
- **Crédibilité** : La fausse piste est-elle suffisamment convaincante ?
- **Fair play** : Le lecteur attentif peut-il déceler la vraie piste malgré le leurre ?
- **Résolution** : La fausse piste est-elle explicitement démontée ou laissée en suspens (défaut) ?
