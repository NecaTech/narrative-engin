# Règle : Foreshadowing (Présage)

## Définition
Indices subtils plantés tôt dans le récit qui préparent et légitiment les développements futurs, créant une attente inconsciente chez le lecteur.

## Principe
Un bon foreshadowing passe inaperçu à la première lecture mais devient évident à la relecture. Il rend les twists "surprenants mais inévitables".

## Types de Foreshadowing
- **Direct** : Prophéties, avertissements explicites.
- **Symbolique** : Objets, météo, couleurs qui annoncent.
- **Dialogue** : Phrases à double sens comprises rétrospectivement.
- **Comportemental** : Actions mineures qui révèlent la vraie nature.

## Application pour l'Audit
- **Préparation** : Les événements majeurs sont-ils annoncés subtilement ?
- **Subtilité** : Les indices sont-ils trop évidents (spoiler) ou trop obscurs (deus ex machina) ?
- **Cohérence rétroactive** : À la relecture, les indices convergent-ils vers la révélation ?
