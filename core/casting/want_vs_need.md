# Règle : Want vs Need (Désir vs Besoin)

## Définition
Chaque personnage principal doit être animé par deux forces distinctes et souvent contradictoires : un Désir conscient (Want) et un Besoin inconscient (Need).

## Les Deux Moteurs

### Want (Désir)
L'objectif visible et déclaré du personnage. Ce qu'il croit vouloir.
- Exemple : Gagner la guerre, trouver le trésor, obtenir la promotion.

### Need (Besoin)
La leçon psychologique ou émotionnelle que le personnage doit apprendre pour évoluer. Ce dont il a réellement besoin.
- Exemple : Apprendre à faire confiance, accepter sa vulnérabilité, pardonner.

## Le Conflit Interne
Le récit devient puissant quand le Want et le Need s'opposent. Le personnage doit souvent renoncer à son Désir pour satisfaire son Besoin (arc de transformation).

## Application pour l'Audit
- **Définition claire** : Le Want et le Need de chaque personnage principal sont-ils explicitement définis dans les specs ?
- **Tension interne** : Le Want et le Need créent-ils un conflit observable dans les actions du personnage ?
- **Arc de transformation** : Le personnage évolue-t-il du Want vers le Need au cours du récit ?
- **Cohérence avec Save the Cat** : Le "Midpoint" et le "All is Lost" exploitent-ils ce conflit interne ?
