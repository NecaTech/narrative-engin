# Règle : Rule of Three (Règle de Trois)

## Définition
Principe selon lequel les informations, événements ou éléments présentés par groupes de trois sont plus satisfaisants, mémorables et efficaces.

## Applications
- **Structure** : Trois actes, trois essais avant réussite.
- **Humour** : Setup, setup, punchline.
- **Rhétorique** : Trois exemples, trois arguments.
- **Suspense** : Deux fausses alertes, une vraie.

## Pourquoi ça fonctionne
- Trois est le minimum pour établir un pattern.
- Le troisième élément peut confirmer ou subvertir les attentes.
- Rythme naturel pour le cerveau humain.

## Application pour l'Audit
- **Patterns** : Les éléments importants sont-ils présentés par trois ?
- **Subversion** : Le troisième élément apporte-t-il un twist ou une confirmation ?
- **Rythme** : Le récit utilise-t-il des triplets pour créer du rythme ?
