# Règle : Le "2 + 2" (Andrew Stanton)

## Définition
Ne donnez pas "4" au lecteur, donnez-lui "2 + 2". Autrement dit, ne fournissez pas la conclusion d'une intrigue directement ; donnez les éléments pour que le lecteur fasse le travail mental lui-même.

## Principe
L'engagement cognitif du lecteur est maximisé lorsqu'il doit assembler les pièces du puzzle. Un lecteur qui "travaille" pour comprendre est un lecteur investi.

## Application pour l'Audit
- **Indices vs Explications** : Le texte contient-il des indices subtils ou des explications directes ?
- **Déduction du lecteur** : Le lecteur peut-il anticiper un retournement s'il est attentif, sans que l'auteur ne le lui dise explicitement ?
- **Satisfaction narrative** : La révélation finale récompense-t-elle le lecteur attentif ?
