# Règle : La Méthode Flocon (Snowflake Method)

## Définition
Un processus de conception narrative qui part du simple vers le complexe, de manière fractale. L'histoire est d'abord une phrase, puis un paragraphe, puis une page, avant de devenir une structure complète.

## Principe
1. **Une phrase** : Résumez votre histoire en une seule phrase (le pitch).
2. **Un paragraphe** : Développez en 5 phrases (setup, 3 désastres, résolution).
3. **Une page** : Étendez chaque phrase du paragraphe.
4. **Les personnages** : Faites de même pour chaque personnage clé.
5. **Le sommaire** : Développez jusqu'à obtenir un synopsis détaillé.

## Application pour l'Audit
- **Cohérence du pitch** : L'histoire finale respecte-t-elle toujours la phrase de départ ?
- **Traçabilité** : Chaque chapitre peut-il être relié à un élément du synopsis initial ?
- **Équilibre** : Les trois "désastres" sont-ils bien répartis dans la structure ?
