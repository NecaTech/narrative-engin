# Règle : La Loi du Contraste de Voix

## Définition
Si vous retirez les noms devant les dialogues, le lecteur doit pouvoir identifier qui parle uniquement par le vocabulaire, le rythme et la syntaxe.

## Principe
Chaque personnage doit avoir une "empreinte vocale" unique :
- **Vocabulaire** : Registre de langue (soutenu, familier, technique).
- **Rythme** : Phrases courtes et percutantes vs longues et sinueuses.
- **Tics de langage** : Expressions récurrentes, hésitations, jurons spécifiques.

## Application pour l'Audit
- **Test de l'anonymat** : Masquer les noms des personnages et vérifier si les répliques restent attribuables.
- **Cohérence interne** : Un personnage change-t-il de registre sans raison narrative ?
- **Distinction** : Deux personnages peuvent-ils être confondus dans un dialogue ?
