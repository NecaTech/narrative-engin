# Règle : La Théorie de l'Iceberg (Sous-texte)

## Définition
Théorisée par Ernest Hemingway. Les personnages ne disent jamais exactement ce qu'ils pensent. Le texte (ce qui est dit) doit être porté par le sous-texte (l'émotion, l'agenda caché, la vérité non-dite).

## Principe
Comme un iceberg, seule une petite partie de la vérité émerge à la surface (le dialogue). La masse immergée (les intentions, les peurs, les désirs) doit être ressentie par le lecteur sans être explicitement énoncée.

## Les Trois Niveaux
1. **Ce que le personnage dit** : Les mots prononcés.
2. **Ce que le personnage pense** : L'intention consciente derrière les mots.
3. **Ce que le personnage ressent vraiment** : L'émotion profonde, souvent inavouée.

## Exemple
- **Texte** : "Ça va, je n'ai pas besoin d'aide."
- **Sous-texte** : Le personnage est désespéré mais refuse de montrer sa vulnérabilité.

## Application pour l'Audit
- **Dialogues plats** : Les personnages disent-ils exactement ce qu'ils pensent, sans filtre ni tension ?
- **Tension implicite** : Le lecteur peut-il percevoir une émotion ou un conflit non exprimé ?
- **Économie de mots** : Le dialogue fait-il confiance à l'intelligence du lecteur pour combler les blancs ?
