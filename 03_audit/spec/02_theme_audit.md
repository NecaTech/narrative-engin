# Protocole : Theme Audit (Audit du Thème - Étape 2/9)

## Objectif
Vérifier que le thème est bien défini, cohérent avec l'intention, et explorable sur la durée d'un récit.

## Points de Contrôle

### Cohérence avec l'Intention
- [ ] Le thème découle-t-il naturellement de l'intention ?
- [ ] Le thème est-il le véhicule approprié pour le message ?
- [ ] Thème et intention pointent-ils dans la même direction ?

### Spécificité
- [ ] Le thème est-il précis (pas juste "l'amour" mais "l'amour sacrificiel") ?
- [ ] L'angle choisi est-il original ou du moins personnel ?
- [ ] Le thème peut-il être exploré sous plusieurs facettes ?

### Question Thématique
- [ ] Y a-t-il une vraie QUESTION morale/philosophique ?
- [ ] Cette question admet-elle plusieurs réponses valides ?
- [ ] Le récit explore-t-il sans prêcher ?

### Ton Émotionnel
- [ ] Le ton est-il défini clairement ?
- [ ] Le ton est-il cohérent avec le genre (cf. étape 3) ?
- [ ] Le ton sert-il le thème ?

## Signaux d'Alerte
- ⚠️ Thème trop vaste ("la vie", "le temps")
- ⚠️ Thème sans question (affirmation dogmatique)
- ⚠️ Thème déconnecté de l'intention
- ⚠️ Ton contradictoire avec le thème
