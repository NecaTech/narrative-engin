# Protocole : Cliche Blacklist (Le Purgatoire des Formules Mortes)

## Connexions Systèmes
- **Règles Liées** : `00_core/style/voice_contrast.md`, `00_core/theme/objective_correlative.md`
- **Audit Parent** : `03_audit/spec/09_voice_audit.md`
- **Usage** : Vérification post-écriture, révision

## Objectif
Identifier et éliminer les **métaphores usées**, les situations prévisibles et les facilités d'écriture qui affaiblissent votre voix unique.

---

## Grille de Scoring (/10)

| Critère | Score | Description |
|---|---|---|
| **Métaphores Fraîches** | /2 | Pas de métaphores mortes |
| **Situations Originales** | /2 | Évitement des tropes éculés |
| **Descriptions Vivantes** | /2 | Pas de descriptions paresseuses |
| **Dialogues Naturels** | /2 | Évitement des répliques clichées |
| **Voix Unique** | /2 | Formulations propres à l'auteur |

---

## Catégories de Clichés à Traquer

### 1. Métaphores Mortes (Kill on Sight)

| ❌ BANNIES | ✅ ALTERNATIVES |
|---|---|
| "Un silence de mort" | [Créer votre propre] |
| "Des yeux comme des lacs" | [Créer votre propre] |
| "Le temps s'arrêta" | [Créer votre propre] |
| "Son cœur battait la chamade" | [Créer votre propre] |
| "Une boule dans la gorge" | [Créer votre propre] |
| "Un frisson lui parcourut l'échine" | [Créer votre propre] |

### 2. Situations Prévisibles

| Cliché | Pourquoi c'est un problème |
|---|---|
| Le mentor qui meurt pour motiver le héros | Trop attendu, grief facile |
| L'ennemi qui révèle son plan au dernier moment | Exposition maladroite |
| Le sauvetage in extremis non préparé | Deus Ex Machina |
| Le personnage "élu" sans raison établie | Facilité lazy |
| L'amnésie comme device narratif | Surexploité |

### 3. Descriptions Paresseuses

| ❌ BANNIE | Pourquoi |
|---|---|
| Yeux "perçants" ou "pénétrants" | Vide de sens |
| Sourire "énigmatique" | Non-description |
| Beauté "à couper le souffle" | Tell, pas Show |
| Silence "pesant" ou "lourd de sens" | Trop facile |

### 4. Dialogues Clichés

| ❌ BANNIE | Effet sur le lecteur |
|---|---|
| "Ce n'est pas ce que tu crois" | Eye-roll |
| "On n'a pas le temps d'expliquer" | Frustration |
| "Tu ne comprends pas" | Faux conflit |
| "Je peux tout expliquer" | Télégraphié |

---

## Action

Chaque cliché détecté doit être remplacé par une **formulation originale propre à votre plume**.

---

## Évolution

> ⚠️ Ce fichier doit **grandir** avec chaque audit. Ajoutez ici les clichés que vous découvrez dans votre propre écriture.

---

## Verdict

| Score | Verdict | Action |
|---|---|---|
| **8-10** | ✅ Voix Originale | Aucune correction |
| **5-7** | ⚠️ Quelques Automatismes | Remplacer les clichés identifiés |
| **0-4** | ❌ Prose Générique | Révision approfondie du style |
