# Protocole : Subtext Density Test (Test de Densité du Sous-texte)

## Objectif
Évaluer si une scène a suffisamment de "poids sous la ligne de flottaison" — si le non-dit est aussi riche que le dit.

## Niveaux de Sous-texte à Évaluer

### Niveau 1 : Dialogue
- [ ] Les personnages disent-ils exactement ce qu'ils pensent ? (Mauvais)
- [ ] Y a-t-il un écart entre les mots et l'intention ?
- [ ] Le lecteur doit-il "lire entre les lignes" ?

### Niveau 2 : Action
- [ ] Les gestes contredisent-ils parfois les paroles ?
- [ ] Les petites actions révèlent-elles des états internes ?
- [ ] Le langage corporel ajoute-t-il une couche de sens ?

### Niveau 3 : Scène
- [ ] La scène a-t-elle un sens littéral ET un sens thématique ?
- [ ] Les objets présents ont-ils une valeur symbolique ?
- [ ] L'environnement reflète-t-il l'état émotionnel ?

## Indicateurs de Pauvreté
- Tout est dit explicitement
- Aucune tension entre apparence et réalité
- Le lecteur n'a rien à déduire
- Les émotions sont nommées plutôt que montrées

## Indicateurs de Richesse
- Tension entre texte et sous-texte
- Ironie dramatique présente
- Symbolisme discret mais présent
- Le lecteur "travaille" pour comprendre
